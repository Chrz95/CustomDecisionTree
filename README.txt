Για το ερώτημα 1 της εργασίας τρέξτε το Python Script Part_1.py. 

- Σε περίπτωση που θέλετε να εκτελέσετε τον αλγόριθμο για όλες τις περιπτώσεις των Impurity Functions,Stopping 
Criteria,Pre-Pruning και Post-Pruning δώστε 1 στην πρώτη ερώτηση που θα σας γίνει. 

- Αν θέλετε να ελέγξετε μια περίπτωση με συγκεκριμένες τιμές παραμέτρων επιλέξτε 0. 

Στην συνέχεια, εισάγετε τον αριθμό των φορών που θέλετε να επαναληφθεί το πείραμα για κάθε περίπτωση. Μεγαλύτερη τιμή
σημαίνει μεγαλύτερη ακρίβεια στις μετρικές απόδοσης (παίρνουμε τον μέσο όρο των επαναλήψεων), αλλά το πρόγραμμα θα 
καθυστερήσει περισσότερο να εκτελεστεί πλήρως.

- Στην περίπτωση που επιλέξατε να δείτε όλες τις περιπτώσεις, στην πρώτη ερώτηση, θα ερωτηθείτε στην συνέχεια για το 
κάτω όριο των accuracies που επιθυμείτε να δείτε.

- Σε περίπτωση που επιλέξατε να δείτε μια συγκεκριμένη περίπτωση, θα σας ζητηθεί να εισάγετε τις τιμές των παραμέτρων.
Συγκεκριμένα θα σας ζητηθει το condition δηλαδή τα stopping criteria, το είδος του impurity, το είδος του pre
pruning, του post-pruning και αν επιλέξατε να γίνει pre-pruning θα σας ζητηθεί μια τιμή για το αντίστοιχο threshold.

Μετά απλώς περιμένετε μέχρι το πρόγραμμα να ολοκληρωθεί, όπου θα ενημερωθείτε με αντίστοιχο μήνυμα στην κονσόλα. Τα
αποτελέσματα είναι στο αρχείο Results_1.txt, το οποίο βρίσκεται στον ίδιο φάκελο με το εκτελέσιμο αρχείο. Σε περίπτωση
που θελήσετε να εκτελέσετε το πρόγραμμα για όλες τις περιπτώσεις η εκτέλεση μπορεί να πάρει ώρες και το αρχείο των
αποτελεσμάτων είναι μεγάλο (Ειδικά αν θελήσετε να δείτε όλα τα accuracies και με πολλες επαναλήψεις). Οπότε σας προτείνουμε
να τρέξετε για μεγάλο αριθμό επαναλήψεων (>=100), τις περιπτώσεις που παραθέτουμε στην αναφορά, για να δείτε τα αποτελέσματα.
